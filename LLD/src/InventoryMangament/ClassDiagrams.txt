     - inventory : Map<String, int>
     - alertConfigs : Map<String, List<AlertConfig>> // maps product IDs to lists of AlertConfig objects

     + Warehouse(id)

     - getAlertsToFire(productId, previousQty, newQty) -> List<AlertListener>

AlertConfig is basically a simple data holder (value object) that groups two things together:

threshold → the stock level limit

listener → the object that should be notified when stock crosses that limit

class AlertConfig :
      - threshold : int
      - listener : AlertListener

      + AlertConfig(threshold, listener) 
      + getThreshold() -> int
      + getListener() -> AlertListener

AlertListener is an interface that defines the contract for handling low-stock notifications.

interface AlertListener :
        + onLowStock(warehouseId, productId, currentQuantity) -> void
